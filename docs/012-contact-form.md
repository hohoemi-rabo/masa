## フォーム項目
```typescript
interface ContactForm {
  name: string;        // 必須
  email: string;       // 必須、メール形式
  phone?: string;      // 任意
  category: string;    // 必須、選択式
  message: string;     // 必須、最小50文字
}
```

## 完了条件
- フォーム送信が成功
- データがSupabaseに保存
- 自動返信メールが届く
