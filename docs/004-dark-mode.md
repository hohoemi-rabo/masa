## カラースキーム
```css
/* ライトモード */
--primary: #0070f3
--secondary: #00d9ff
--background: #ffffff
--foreground: #1a1a1a

/* ダークモード */
--primary: #0098ff
--secondary: #00d9ff
--background: #0a0a0a
--foreground: #fafafa
```
